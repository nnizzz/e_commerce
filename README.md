A modern Flutter Food App UI that displays restaurants and fast food items using ListView and GridView. 
This project focuses on UI building, navigation, scrolling layouts, and reusable widgets.
Screens Included:
Intro Screen,
Home Screen,
Restaurants section,
Fast Foods section,
Food Items Screen,
Grid view of food items with images,
Detail Screen .

Key Widgets Used:
Scaffold,
AppBar,
SingleChildScrollView,
ListView.builder,
GridView.builder,
Card,
Navigator.push,
Image.network,
Row, Column, Padding, SizedBox.
